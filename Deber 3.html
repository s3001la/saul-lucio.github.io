Ejercicio 3: Suma de dos números

📌 Crea una función que reciba dos números y retorne su suma.

def suma(a, b): return a + b a = int(input("Ingresa el primer número: ")) b = int(input("Ingresa el segundo número: ")) print("La suma es:", suma(a, b))

Ejercicio 4: Longitud de una palabra

📌 Crea una función que reciba una palabra y retorne su longitud (número de caracteres).

def longitud(palabra): return len(palabra) palabra = input("Ingresa una palabra: ") print("La longitud es:", longitud(palabra))

Ejercicio 6: Mayor de tres números

📌 Crea una función que reciba tres números y retorne el mayor de ellos.

def mayorTres(a, b, c): mayor = a if b > mayor: mayor = b if c > mayor: mayor = c return mayor a = int(input("Ingresa el 1er número: ")) b = int(input("Ingresa el 2do número: ")) c = int(input("Ingresa el 3er número: ")) print("El número mayor es:", mayorTres(a, b, c))

Ejercicio 9: Número positivo o negativo

📌 Crea una función que retorne True si el número es positivo, False en caso contrario.

def esPositivo(numero): return numero > 0 num = float(input("Ingrese un número: ")) if esPositivo(num): print("El número es positivo.") else: print("El número es negativo o cero.")

Ejercicio 10: Invertir cadena

📌 Crea una función que reciba una palabra y retorne la cadena invertida.

def invertirCadena(palabra): return palabra[::-1] palabra = input("Ingrese una palabra: ") print(palabra, "invertida es:", invertirCadena(palabra))
