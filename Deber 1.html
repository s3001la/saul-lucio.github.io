Ejercicio 1: Número mayor o menor que 50

📌 Pide un número al usuario y muestra si es mayor o igual que 50, o menor.

usuario = input("Ingrese su nombre: ") numero = int(input("Ingrese un número: ")) if numero >= 50: print(usuario + ", el número ingresado es mayor o igual que 50") else: print(usuario + ", el número ingresado es menor que 50")

Ejercicio 2: Edad para votar

📌 Pide la edad de una persona y determina si puede votar (18 años o más).

nombre = input("Ingrese su nombre: ") edad = int(input("Ingrese su edad: ")) if edad >= 18: print(nombre + ", es mayor de edad y puede votar") else: print(nombre + ", es menor de edad y NO puede votar")

Ejercicio 3: Número múltiplo de 3

📌 Pide un número y verifica si es múltiplo de 3.

numero = int(input("Ingrese un número: ")) if numero % 3 == 0: print(f"{numero} es múltiplo de 3.") else: print(f"{numero} NO es múltiplo de 3.")

Ejercicio 4: Contraseña correcta

📌 Pide una contraseña y comprueba si es igual a "python123".

contrasena = input("Ingrese su contraseña: ") if contrasena == "python123": print("Contraseña correcta.") else: print("Contraseña incorrecta.")

Ejercicio 6: Número mayor

📌 Pedir dos números y muestra cuál de los dos es mayor.

print("Comparación de 2 números") num1 = float(input("Ingresa el primer número: ")) num2 = float(input("Ingresa el segundo número: ")) if num1 > num2: print(f"El número mayor es: {num1}") elif num2 > num1: print(f"El número mayor es: {num2}") else: print("Ambos números son iguales")

Ejercicio 7: Positivo, negativo o cero

📌 Pide un número y determina si es positivo, negativo o cero.

print("Determinar signo del número") numero = float(input("Ingresa un número: ")) if numero > 0: print("El número es positivo") elif numero < 0: print("El número es negativo") else: print("El número es cero")

Ejercicio 8: Vocal o consonante

📌 Pide una letra e indica si es una vocal.

print("Verificar si es vocal o consonante") letra = input("Ingresa una letra: ").lower() if len(letra) == 1 and letra.isalpha(): if letra in 'aeiou': print(f"'{letra}' es una vocal.") else: print(f"'{letra}' es una consonante.") else: print("Por favor, ingrese una letra válida.")

Ejercicio 9: ¿Hace frío?

📌 Pide una temperatura y muestra si hace frío (menos de 15°C).

print("TEMPERATURA") temp = float(input("Ingrese una temperatura en °C: ")) if temp < 15: print(f"Hace frío: {temp}°C") else: print(f"No hace frío: {temp}°C")

Ejercicio 10: Año bisiesto

📌 Pide un año e indica si es bisiesto usando únicamente condiciones.

print("AÑO BISIESTO") year = int(input("Ingrese un año: ")) if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0): print(f"El año {year} es bisiesto") else: print(f"El año {year} no es bisiesto")
